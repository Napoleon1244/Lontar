      unduh buku

      teks terjemahan Bahasa Indonesia (atau teks asli jika belum diterjemahkan).

      pengaturan
      A 15px

      default: Google Translate (gratis, tanpa kunci). untuk kualitas terjemahan sastra lebih baik, masukkan kunci Gemini Flash — disimpan hanya di browser-mu.

      kunci gratis: aistudio.google.com/apikey

      tentang lontar

      lontar adalah pembaca Project Gutenberg dengan terjemahan otomatis ke Bahasa Indonesia.

      masukkan ID atau link buku dari Gutenberg. lontar ambil teksnya, terjemahkan, lalu sajikan untuk dibaca.

      buku yang sudah diterjemahkan di-cache di browser — buka lagi langsung tanpa proses ulang.

      semua data tersimpan di browser-mu. tidak ada server, tidak ada tracking, tidak ada akun.

      mesin terjemahan:

      · Google Translate — gratis, tanpa kunci, default
      · Gemini Flash — kualitas lebih baik untuk sastra, butuh kunci API gratis
